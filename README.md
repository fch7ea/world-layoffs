# World Layoffs Dataset

This project cleans a world layoffs dataset with over 2300 entries for future data exploration.

## Details

Some of the SQL keywords and functions used are as follows:
* CREATE TABLE
* INSERT
* PARTITION BY
* OVER
* ORDER BY
* DELETE
* UPDATE
* ALTER TABLE
* MODIFY COLUMN
* DROP COLUMN
* JOIN
* ROW_NUMBER()
* TRIM()
* LIKE
* STR_TO_DATE()
